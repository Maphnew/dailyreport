# 금호섬유공업 Daily Report

1. nodejs 설치
2. npm 설치
3. node_module 설치
```bash
$ npm install
```  
4. .env 파일작성(root 폴더)
- 예시  
```
MAIL_ID = 'your.mail@gmail.com'
MAIL_PASS = 'your.mail.password'

DB_HOST=192.168.100.100
DB_PORT=3306
DB_USER=root
DB_PASS=passwod
DB_DATABASE=UYeG_KUMHO
```
5. 실행
- dailyreport.bat 파일 실행 또는
```bash
$ node app.js
```
